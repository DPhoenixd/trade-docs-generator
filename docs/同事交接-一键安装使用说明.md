# P.I/P.L一点腾 同事交接使用说明

## 最简单的使用方式

1. 解压 `P.I-P.L一点腾-便携版.zip`
2. 打开解压后的文件夹
3. 双击 `P.I-P.L一点腾.exe`
4. 浏览器会自动打开工具页面

不需要打开命令行，不需要安装 Python，不需要安装 Node.js。

## 每次做一份新订单

1. 第 1 步上传客户截图，检查颜色、公司色号、数量、单位、单价是否正确
2. 第 2 步输入 PO NO.、Buyer、买方地址、交货期、目的港，并检索面料编号
3. 第 3 步审查金额、KG 单价、Yard、码价、付款方式
4. 先生成 P.I；P.L 需要另外上传或识别码单细码后再生成

## 数据来源说明

- 客户截图：用于生成 P.I 明细
- 面料数据库：用于匹配成分、克重、幅宽、量化、纸筒空差
- P.I 模板：只提供 Excel 格式，不覆盖已填写订单
- P.L 模板/码单：用于 Packing List

## 常见问题

- 如果双击后没有打开页面：等 10 秒，再双击一次
- 如果浏览器提示打不开：确认电脑没有安全软件拦截 exe
- 如果生成失败：检查是否上传了 P.I 模板、是否选择了面料编号、关键字段是否填完
- 如果面料编号查不到：说明当前内置数据库没有这款面料，需要更新数据库文件
